## Quick Start

### Authentication

Compyle uses API tokens for authentication. Include your token in the Authorization header:

```bash
Authorization: Bearer YOUR_API_TOKEN
```

### Base URL

```bash
https://api.compyle.ai/v1
```

### Example Request

```bash
curl -X GET "https://api.compyle.ai/v1/projects" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json"
```

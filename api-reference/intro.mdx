## Planned Endpoints

### Tasks

Manage and track tasks:

- `GET /api/v1/tasks` - List all tasks
- `POST /api/v1/tasks` - Create new task
- `GET /api/v1/tasks/{id}` - Get task details
- `PATCH /api/v1/tasks/{id}` - Update task
- `DELETE /api/v1/tasks/{id}` - Delete task

### Projects

Manage projects:

- `GET /api/v1/projects` - List all projects
- `POST /api/v1/projects` - Create project
- `GET /api/v1/projects/{id}` - Get project details
- `PATCH /api/v1/projects/{id}` - Update project
- `GET /api/v1/projects/{id}/tasks` - List project tasks

### Artifacts

Retrieve task artifacts:

- `GET /api/v1/tasks/{id}/artifacts/research` - Get research.md
- `GET /api/v1/tasks/{id}/artifacts/planning` - Get planning.md
- `GET /api/v1/tasks/{id}/artifacts/pr` - Get pull request info

### Repositories

Manage connected repositories:

- `GET /api/v1/repositories` - List connected repos
- `POST /api/v1/repositories` - Connect new repo
- `DELETE /api/v1/repositories/{id}` - Disconnect repo

### Agents

Monitor agent status:

- `GET /api/v1/agents/{task_id}` - Get agent status
- `GET /api/v1/agents/{task_id}/logs` - View agent logs

### Webhooks (Coming Soon)

Receive events about task changes:

- `POST /api/v1/webhooks` - Create webhook
- `GET /api/v1/webhooks` - List webhooks
- `DELETE /api/v1/webhooks/{id}` - Delete webhook

## Rate Limiting

Rate limits apply per plan tier:

| Plan | Requests/minute | Concurrent |
|------|-----------------|-----------|
| Starter | 30 | 1 |
| Pro | 100 | 3 |
| Pro+ | 300 | 10 |

**Backoff Strategy**: Use exponential backoff when rate limited

## Error Handling

All endpoints return standard error responses:

```json
{
  "error": "validation_error",
  "message": "Task title is required",
  "status": 400,
  "request_id": "req_123456"
}
```

Common status codes:
- `200` - Success
- `400` - Bad request
- `401` - Unauthorized
- `403` - Forbidden
- `404` - Not found
- `429` - Rate limited
- `500` - Server error

## Code Examples

### JavaScript Example

```javascript
const CompyleAPI = require('compyle-api');

const client = new CompyleAPI({
  apiKey: 'your-api-key'
});

// Create a task
const task = await client.tasks.create({
  projectId: 'proj_123',
  title: 'Add user authentication',
  description: 'Implement OAuth login',
  repositoryId: 'repo_456'
});

console.log(task.id);
```

### Python Example

```python
from compyle import CompyleClient

client = CompyleClient(api_key='your-api-key')

# List all tasks
tasks = client.tasks.list()

for task in tasks:
    print(f"{task.id}: {task.title}")
```

### cURL Example

```bash
# Create a task
curl -X POST https://api.compyle.ai/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "projectId": "proj_123",
    "title": "Add user authentication",
    "description": "Implement OAuth login"
  }'
```

## Version History

### Current Version: v1 (Beta)

**Breaking Changes**: None yet (still in beta)

### API Versioning Strategy

- New features added under current version
- Breaking changes trigger major version bump (v2, v3, etc.)
- Deprecation warnings issued 6 months before breaking changes
- Old versions maintained for 12 months

## SDK Roadmap

### JavaScript/TypeScript SDK (Q1 2026)

```typescript
import { CompyleClient } from '@compyle/sdk';

const client = new CompyleClient({ apiKey });
const task = await client.tasks.create({...});
```

### Python SDK (Q1 2026)

```python
from compyle import CompyleClient

client = CompyleClient(api_key='...')
task = client.tasks.create(...)
```

### Go SDK (Q1 2026)

```go
client, _ := compyle.NewClient(apiKey)
task, _ := client.Tasks.Create(ctx, req)
```

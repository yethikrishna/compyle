**Features**:
- ✓ Question-driven planning workflow
- ✓ Research phase with codebase analysis
- ✓ Planning phase with clarifying questions
- ✓ Implementation phase with real-time validation
- ✓ Custom rules engine
- ✓ GitHub integration
- ✓ Free beta access
- ✓ API (limited beta)

**Documentation**:
- Complete user documentation
- Getting Started guides
- Feature deep dives
- API reference
- FAQ and troubleshooting

**UI/UX**:
- Dashboard for task management
- Real-time task progress monitoring
- Settings and configuration interface
- Command palette for quick actions
- Notification system

### Known Limitations

- Single-user workspaces only (team features coming Q1 2026)
- No Slack/Jira integrations yet
- Limited API availability
- Beta rate limiting
- Research phase required (skip coming soon)

## Versioning Policy

We follow Semantic Versioning (MAJOR.MINOR.PATCH):

- **MAJOR**: Breaking changes (v1.0 → v2.0)
- **MINOR**: New features, backward compatible
- **PATCH**: Bug fixes, backward compatible

**Deprecation Policy**: 6 months warning before breaking changes

---

## End of Life

We maintain versions for 12 months minimum after release.

    - **Research:** Automatic codebase analysis
    - **Planning:** Interactive clarifying questions
    - **Implementation:** Real-time code generation with validation

    **GitHub Integration:**

    - OAuth authentication
    - Automatic PR creation
    - Branch management
    - Multi-repository support

    **Custom Rules System:**

    - JSON-based pattern configuration
    - Real-time enforcement during implementation

    - Multi-repo task tracking
    - Activity monitoring
    - Credit usage tracking

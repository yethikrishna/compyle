# Command Palette

## What is the Command Palette?

The Command Palette is a quick-access menu for Compyle features. Press **`Cmd+K`** (Mac) or **`Ctrl+K`** (Windows/Linux) from anywhere to open it.

Use it to:
- Create new tasks
- View task status
- Access settings
- Navigate documentation
- Search features

## Available Commands

### Project Commands

| Command | Shortcut | Description |
|---------|----------|-------------|
| New Task | `Cmd+Shift+N` | Create a new task |
| View Tasks | `Cmd+Shift+T` | List all tasks in project |
| Project Settings | `Cmd+,` | Open project settings |
| Connected Repos | `Cmd+Shift+R` | Manage connected repositories |

### Navigation Commands

| Command | Description |
|---------|-------------|
| Go to Dashboard | Navigate to main dashboard |
| Go to Projects | View all projects |
| Go to Settings | Open account settings |
| Go to Documentation | Open help docs |

### Task Commands

| Command | Description |
|---------|-------------|
| View Task Status | Check current task progress |
| View Research Phase | See research.md |
| View Planning Phase | See planning.md |
| Download Artifacts | Export task documents |

### System Commands

| Command | Description |
|---------|-------------|
| Toggle Dark Mode | Switch theme |
| Help | Show command reference |
| About | About Compyle |
| Support | Contact support |

## Using the Command Palette

<Steps>
  <Step title="Open Palette">
    Press **Cmd+K** (Mac) or **Ctrl+K** (Windows/Linux)
  </Step>
  <Step title="Type Command">
    Start typing the command you want
  </Step>
  <Step title="Select Result">
    Use arrow keys or mouse to select
  </Step>
  <Step title="Execute">
    Press Enter or click to run
  </Step>
</Steps>

## Quick Tips

<Tip title="Search Across All Commands">
  Type partial words. "new" finds "New Task", "new project", etc.
</Tip>

<Tip title="View Keyboard Shortcuts">
  Type "?" or "help" to see all shortcuts
</Tip>

<Tip title="Fuzzy Matching">
  "nt" matches "new task", "ntp" matches "new task project"
</Tip>

<Tip title="Recent Commands">
  Frequently used commands appear at top
</Tip>

## Common Workflows

### Create and Launch Task

```
1. Cmd+K
2. Type "new task"
3. Select "New Task"
4. Fill in task details
5. Launch
```

### Check Task Progress

```
1. Cmd+K
2. Type "task status"
3. Select task from list
4. View current phase and progress
```

### Access Settings

```
1. Cmd+K
2. Type "settings"
3. Select section (project, account, etc.)
```

# Integrations

Connect Compyle with tools you already use to enhance your workflow and unlock more automation possibilities.

## What Are Integrations?

Integrations connect Compyle with third-party services and tools, enabling:
- **Richer context** - Pull information from external systems
- **Better communication** - Get updates in your preferred channels
- **More automation** - Trigger tasks from other tools
- **Seamless workflows** - Keep your existing toolchain

## Integration Guides

### GitHub <span id="github"></span>

GitHub integration is essential for Compyle. It enables:
- Reading your code and understanding architecture
- Creating branches for task implementation
- Submitting pull requests with changes
- Validating against your custom rules

**Setup**: Connected during account creation. Manage in Settings → Connected Repositories.

### Anthropic API <span id="api"></span>

Use your own Anthropic API key to:
- Control your own API costs
- Set custom rate limits
- Maintain data privacy
- Use specific model versions

<Steps>
  <Step title="Get Your API Key">
    Visit [Anthropic Console](https://console.anthropic.com) and generate an API key
  </Step>
  <Step title="Go to Account Settings">
    Click your profile → Settings → Integrations
  </Step>
  <Step title="Add API Key">
    Paste your Anthropic API key in the "Custom LLM Provider" field
  </Step>
  <Step title="Save and Verify">
    Compyle will verify the key works before saving
  </Step>
</Steps>

<Warning>
  **Protect Your API Keys**

  Never share your API keys publicly. Store them securely in environment variables or a password manager. If compromised, regenerate immediately in the Anthropic console.
</Warning>

## API Key Management

<Steps>
  <Step title="Go to Account Settings">
    Click your profile icon → Settings
  </Step>
  <Step title="Select "API Keys"">
    View all your generated keys
  </Step>
  <Step title="Generate New Key">
    Click "Create New Key" to generate additional keys
  </Step>
  <Step title="Copy Key">
    Keys are shown only once. Copy and store securely immediately.
  </Step>
  <Step title="Regenerate or Revoke">
    Invalidate compromised keys by regenerating or deleting
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="My API key isn't working">
    - Verify the key is active in Anthropic console
    - Ensure you have remaining quota
    - Check for typos when pasting
    - Regenerate a new key and try again
  </Accordion>
  <Accordion title="GitHub integration failed">
    - Ensure you're logged into GitHub
    - Check you have admin access to your repositories
    - Disconnect and reconnect GitHub access
    - Contact support if issues persist
  </Accordion>
  <Accordion title="How do I disconnect an integration?">
    Go to Settings → Integrations → [Service Name] → "Disconnect". Changes take effect immediately.
  </Accordion>
</AccordionGroup>

Quick Start

Get up and running with Compyle in minutes

Installation

Follow these steps to install Compyle on your system.

Prerequisites

Before installing Compyle, ensure you have the following:

  • Node.js v14 or higher
  • npm v6 or higher or yarn v1.22 or higher
  • Git for version control

Installing Compyle

You can install Compyle using npm or yarn:

npm 
npm install -g compyle-cli
yarn 
yarn global add compyle-cli
pnpm 
pnpm add -g compyle-cli

Verify the installation:

bash 
compyle --version

First Project

Create your first Compyle project in seconds.

bash 
compyle init my-project

This will create a new directory my-project with the basic project structure:

directory 
my-project/
├── compyle.config.js
├── src/
│   ├── components/
│   │   └── index.js
│   ├── pages/
│   │   └── index.js
│   └── index.js
├── package.json
└── README.md

Basic Usage

Learn how to use Compyle with these basic commands.

Development Server

Start a local development server with hot reloading.

compyle dev

Build for Production

Build your project for production deployment.

compyle build

Preview Build

Preview your production build locally.

compyle preview

Configuration

Configure Compyle using the compyle.config.js file.

javascript 
module.exports = {
  // Project name
  name: 'my-project',
  
  // Output directory
  output: 'dist',
  
  // Development server settings
  devServer: {
    port: 3000,
    open: true
  },
  
  // Build settings
  build: {
    minify: true,
    sourcemap: process.env.NODE_ENV !== 'production'
  },
  
  // Plugins
  plugins: [
    // Add your plugins here
  ]
};
